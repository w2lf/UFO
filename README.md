# UFO - Unreal Engine Project

A UFO game project built with Unreal Engine.

## Getting Started

### Prerequisites
- Unreal Engine 5.x
- Visual Studio 2022 (for C++ development)

### Setup
1. Clone this repository
2. Right-click `UFO.uproject` → **Generate Visual Studio project files**
3. Open `UFO.sln` in Visual Studio
4. Build the solution (Development Editor)
5. Open `UFO.uproject` in Unreal Engine

## Project Structure

```
UFO/
├── Source/        # C++ source code
├── Content/       # Game assets (meshes, textures, blueprints)
├── Config/        # Unreal Engine configuration files
├── Shaders/       # Custom shaders
└── UFO.uproject   # Unreal project file
```

## Build

Run `build.bat` to build the project from the command line.
